# §14. UI レイアウト

← [仕様書インデックス](./README.md)

---

## 14.1 PC版
- **チャート最大化レイアウト**: サイドバーは銘柄リスト / インジケーター切替 / 描画ツール / TradePanel / アクションボタンに絞る
- **メモはサイドバーに常駐しない**: チャートとメモを同時表示すると縦空間が狭くなるため、**メモパネル切替式**([§7.3](./07-memo.md#73-ui-メモパネル切替式))を採用する
  - ヘッダー右端の「📝 メモ」ボタン / `M` キー / `Esc` で開閉
  - 開くとチャート領域を覆う全画面オーバーレイに「銘柄別メモ + 横断メモ」の 2 段
- **エントリー組み立て** はサイドバー下部の TradePanel に集約([§7.4](./07-memo.md#74-エントリー時の必須項目最小限)): 現在値表示 / SL・TP の配置状況 / 方向自動表示バッジ / 単一の「エントリー」ボタン。価格や方向の手入力欄は持たない(SL/TP はチャート上クリックで配置)
- 見送り操作は同じサイドバーの「見送り」ボタンから(理由テキストは任意)

## 14.2 スマホ版
- チャート全画面表示
- メモパネルは全画面モーダルで表示(PC 版と同じ構造、タップで開閉)
- 縦持ち・横持ち両対応
